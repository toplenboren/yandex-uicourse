История JS

История JS началась с NetScape, Веб должен стать более динамичным. Захотелось создать скриптовый язык, которым легко бы могли овладеть много людей

Среди кандидатов были:

  • Java
  • Python
  • Tcl
  • Scheme (МТ привет)
  • Mocha -> LiveScript -> JavaScript

Появился Брендон Айк, сказал, что язык должен быть написан за 10 дней, потому что иначе язык стал бы слишком сложным. Плюс Брендон боялся гигантов рынка. А ещё он сказал, что JS должен выглядеть как Java

Why JS?
  • Простота и гибкость
  • Разработан с учетом требований веба
  • Сиподобный синтаксис
Что в итоге:
  • Язык сделали за 10 дней
  • Наследие и баги из Java
  • Появилось куча движков JS, лидер - V8
